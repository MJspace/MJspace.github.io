-->따라서 일반 텍스트로 서식이 있는 문서를 작성할 수 있다!

# 마크다운 문법
## Header
- '#'의 개수로 글자의 크기를 조정해 제목을 작성할 수 있다.
## Italic
- '* *, _ _'로 기울임체 작성 가능
## Bold
- '** **, __ __'로 강조체 작성 가능
## Strikethrough
- '~ ~'로 취소선 작성 가능
## Unordered List
- '-, *'으로 순서없는 리스트 작성 가능
## Ordered List
- '숫자 ex)1.'로 순서없는 리스트 작성 가능
## Code
- ' '로 코드 작성
## Code Block
- ''' '''으로 코드블럭 작성

```python
def power(x: int):
  return x ** 2
```
