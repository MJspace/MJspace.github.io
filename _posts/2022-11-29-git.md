# Git
:분산 버전관리 시스템
->git <명령어> 를 통해 관리, git의 저장단위는 commit!
 ## Git 명령어
 - git init -> 로컬 저장소 생성
 - git status -> 현재 깃 상태 확인
 - git add -> 스테이징 영역에 파일 추가 됨
 - git commit
 - git push
 - git reset
 - git pull
 - git log -> commit 기록 확인

## Git branch
- git brach <> - 브랜치 생성
- git checkout <> -브랜치 전환
- git merge<> -브랜치 병합
- git branch -d <> -브랜치 삭제


# Github
->로컬 저장소를 넘어 원격 저장소로!
 깃허브는 원격저장소로 분산 버전 관리 툴인 깃 저장소 호스팅을 지원하는 웹 서비스이다.
